🔐 MCP Server OAuth Test

This tool helps you test your MCP server with proper OAuth tokens matching the configuration in useAutoOAuth.ts.

📋 Configuration

🚀 Testing Steps

📝 Instructions

  1. Enter your Google OAuth Client ID - This should be the same one used in your Looker extension
  2. Click "Get OAuth Token" - This will open Google OAuth with the correct parameters
  3. Complete OAuth flow - Sign in and authorize the application
  4. Extract the token - Copy the id_token from the redirect URL
  5. Test the server - Use the token to test your MCP server

📋 OAuth Parameters Used

This tool uses the exact same parameters as useAutoOAuth.ts:

  • scope: "openid email profile"
  • response_type: "id_token"
  • nonce: <random>
